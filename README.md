# Yolo V5 for Nvidia Jetson Xavier
해당 Repository는 Nvidia Jetson Xavier AGX 보드에서 yolov5를 동작하고 객체 검출값을 화면에 출력하는 리포지토리 입니다.

## 개발환경
- __Hardware__
  - Nvidia Jetson Xavier AGX Board
  - Raspberry Pi Camera
- __Software__
  - jetpack 4.4
  - python 3.6
  - torch 1.7.0
  - torchvision 0.8.0
